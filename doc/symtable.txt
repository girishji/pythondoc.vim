"symtable" — Access to the compiler’s symbol tables
***************************************************

**Source code:** Lib/symtable.py

======================================================================

Symbol tables are generated by the compiler from AST just before
bytecode is generated.  The symbol table is responsible for
calculating the scope of every identifier in the code.  "symtable"
provides an interface to examine these tables.


Generating Symbol Tables
========================

                                            *symtable()..symtable.txt*
symtable.symtable(code, filename, compile_type)

   Return the toplevel "SymbolTable" for the Python source _code_.
   _filename_ is the name of the file containing the code.
   _compile_type_ is like the _mode_ argument to "compile()".


Examining Symbol Tables
=======================

class symtable.SymbolTable                 *SymbolTable..symtable.txt*

   A namespace table for a block.  The constructor is not public.

   get_type()                      *SymbolTable.get_type()..symtable.txt*

      Return the type of the symbol table.  Possible values are
      "'class'", "'module'", and "'function'".

   get_id()                          *SymbolTable.get_id()..symtable.txt*

      Return the table’s identifier.

   get_name()                      *SymbolTable.get_name()..symtable.txt*

      Return the table’s name.  This is the name of the class if the
      table is for a class, the name of the function if the table is
      for a function, or "'top'" if the table is global ("get_type()"
      returns "'module'").

   get_lineno()                  *SymbolTable.get_lineno()..symtable.txt*

      Return the number of the first line in the block this table
      represents.

   is_optimized()              *SymbolTable.is_optimized()..symtable.txt*

      Return "True" if the locals in this table can be optimized.

   is_nested()                    *SymbolTable.is_nested()..symtable.txt*

      Return "True" if the block is a nested class or function.

   has_children()              *SymbolTable.has_children()..symtable.txt*

      Return "True" if the block has nested namespaces within it.
      These can be obtained with "get_children()".

   get_identifiers()        *SymbolTable.get_identifiers()..symtable.txt*

      Return a view object containing the names of symbols in the
      table. See the documentation of view objects.

   lookup(name)                      *SymbolTable.lookup()..symtable.txt*

      Lookup _name_ in the table and return a "Symbol" instance.

   get_symbols()                *SymbolTable.get_symbols()..symtable.txt*

      Return a list of "Symbol" instances for names in the table.

   get_children()              *SymbolTable.get_children()..symtable.txt*

      Return a list of the nested symbol tables.

class symtable.Function                       *Function..symtable.txt*

   A namespace for a function or method.  This class inherits
   "SymbolTable".

   get_parameters()             *Function.get_parameters()..symtable.txt*

      Return a tuple containing names of parameters to this function.

   get_locals()                     *Function.get_locals()..symtable.txt*

      Return a tuple containing names of locals in this function.

   get_globals()                   *Function.get_globals()..symtable.txt*

      Return a tuple containing names of globals in this function.

   get_nonlocals()               *Function.get_nonlocals()..symtable.txt*

      Return a tuple containing names of nonlocals in this function.

   get_frees()                       *Function.get_frees()..symtable.txt*

      Return a tuple containing names of free variables in this
      function.

class symtable.Class                             *Class..symtable.txt*

   A namespace of a class.  This class inherits "SymbolTable".

   get_methods()                      *Class.get_methods()..symtable.txt*

      Return a tuple containing the names of methods declared in the
      class.

class symtable.Symbol                           *Symbol..symtable.txt*

   An entry in a "SymbolTable" corresponding to an identifier in the
   source.  The constructor is not public.

   get_name()                           *Symbol.get_name()..symtable.txt*

      Return the symbol’s name.

   is_referenced()                 *Symbol.is_referenced()..symtable.txt*

      Return "True" if the symbol is used in its block.

   is_imported()                     *Symbol.is_imported()..symtable.txt*

      Return "True" if the symbol is created from an import statement.

   is_parameter()                   *Symbol.is_parameter()..symtable.txt*

      Return "True" if the symbol is a parameter.

   is_global()                         *Symbol.is_global()..symtable.txt*

      Return "True" if the symbol is global.

   is_nonlocal()                     *Symbol.is_nonlocal()..symtable.txt*

      Return "True" if the symbol is nonlocal.

   is_declared_global()       *Symbol.is_declared_global()..symtable.txt*

      Return "True" if the symbol is declared global with a global
      statement.

   is_local()                           *Symbol.is_local()..symtable.txt*

      Return "True" if the symbol is local to its block.

   is_annotated()                   *Symbol.is_annotated()..symtable.txt*

      Return "True" if the symbol is annotated.

      New in version 3.6.

   is_free()                             *Symbol.is_free()..symtable.txt*

      Return "True" if the symbol is referenced in its block, but not
      assigned to.

   is_assigned()                     *Symbol.is_assigned()..symtable.txt*

      Return "True" if the symbol is assigned to in its block.

   is_namespace()                   *Symbol.is_namespace()..symtable.txt*

      Return "True" if name binding introduces new namespace.

      If the name is used as the target of a function or class
      statement, this will be true.

      For example:
>
         >>> table = symtable.symtable("def some_func(): pass", "string", "exec")
         >>> table.lookup("some_func").is_namespace()
         True
<
      Note that a single name can be bound to multiple objects.  If
      the result is "True", the name may also be bound to other
      objects, like an int or list, that does not introduce a new
      namespace.

   get_namespaces()               *Symbol.get_namespaces()..symtable.txt*

      Return a list of namespaces bound to this name.

   get_namespace()                 *Symbol.get_namespace()..symtable.txt*

      Return the namespace bound to this name. If more than one or no
      namespace is bound to this name, a "ValueError" is raised.
